Request Processing subflow for the Service Access from WebSphere MQ: one-way pattern

This subflow has, by default, no impact. When the Service Access from WebSphere MQ: one-way pattern is used to generate a pattern instance, it contains only a Passthrough node. This subflow is included to provide a place where a pattern instance can be customized by using standard tooling, without changing the structure of the pattern.
