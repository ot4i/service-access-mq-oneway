Sample resources for the Service Access from WebSphere MQ: one-way pattern

The Service Access from WebSphere MQ: one-way pattern provides a set of sample resources. The sample resources include a message set and a deployable WSDL. The sample resources help you start to use this pattern.

Note: You can use the links in the information center to import and remove the sample resources only when you use the information center that is integrated with the IBM Integration Toolkit.
